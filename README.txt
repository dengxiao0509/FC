Utilisation de l'outil sur Xampp:

1.	Décompresser le répertoire ver1_0.zip et mettre tous ses sous-répertoires(/php,/images/, ...) dans le répertoire  /xampp/htdocs	(comme: /xampp/htdocs/php);

2. 	Démarrer Apache et MySQL dans le Xampp Control Panel;

3.	Saisir http://localhost/php/cloudcoach.php dans le URL bar du navigateur (Chrome ou Firefox).



• Le fichier de configuration :  /php/include/cloudcoach_configuration.php
• Les fichiers (Pièces jointes)  téléchargés sont dans le répertoire /php/files (pas besoin de créer ce répertoire manuellement)



Autres information voir le document "Xiao_Evolution_v1.0.docx".
